# 🚀 Basic Full Stack App

A simple Full Stack application built while learning how frontend and backend communicate with each other.

This project may be small, but it helped me understand some of the most important concepts in web development:

- React + Vite frontend setup
- Express.js backend setup
- Creating API routes
- Sending data from backend to frontend
- Working with JSON responses
- Project structure for full stack applications
- Environment variables with dotenv
- Running frontend and backend separately

## 🚀 Running The Project

### Clone Repository

```bash
git clone <your-repository-url>
```

### Backend

```bash
cd backend
npm install
npm start
```

Server runs on:

```bash
http://localhost:3000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```
